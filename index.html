Movie and TV Show Tracker Application

Skills used: C#, MySQL, Integration of API, Winforms Development

Overview:
 Using C# Winforms and the .NET framework I have created a desktop Windows application designed for effortlessly tracking movies and TV shows. Utilizing The Movie Database (TMDB) API, users can seamlessly search for their desired content and access comprehensive details. They can add chosen titles to their personal list, which is stored using a MySQL database. Moreover, users can mark items as watched, remove them from the list, and even incorporate custom content of their own.
